# POC do processo de migração dos dados dos bancos de dados.


## Objetivo

Baseado nos código já criados para a migração dos dados dos bancos de dados, criar um POC executando o código como "DRY-RUN".
Durante a execução gerar dados de todas as ocorrências de migração, como por exemplo, quais dados seriam migrados, quais dados não seriam migrados, quais dados seriam migrados com erros, etc.

Objetivo prinicipal é encontrar todas as ocorrências possíveis durante a migração dos dados dos bancos de dados, para que possamos criar um processo de migração mais robusto e confiável, evitando assim, problemas futuros durante a migração dos dados dos bancos de dados.

## Escopo

- Criar um POC executando o código de migração dos dados dos bancos de dados como "DRY-RUN".
- Gerar amostrar de 10 casos de cada ocorrência de migração. Exemplo, quais dados seriam migrados, quais dados não seriam migrados, quais dados seriam migrados com erros, etc.
- No final da execução gerar um report com os dados gerados e a análise das ocorrências.
